# Battery Voltage Monitoring

A simple C++ console program that repeatedly reads battery voltage values and reports whether they are within the normal range.

## Description
The program prompts the user for a battery voltage reading. If the voltage is 12.0 V or higher, it reports the battery as normal and asks for the next reading. When a reading drops below 12.0 V, the program stops and displays a low-voltage warning.

- `Question_10- Battery Voltage Monitoring.cpp` — main C++ source code
- `Makefile.win` — Windows Makefile generated by Dev-C++


Or use the provided `Makefile.win` if `make` is installed.

## Run
```powershell
."BatteryVoltageMonitor.exe"
```

## Usage
1. Enter a battery voltage when prompted.
2. Continue entering values while they remain at or above 12.0 V.
3. The program ends and warns when a value below 12.0 V is entered.

## Notes
- The program uses `double` to support fractional voltage readings.
- The normal battery threshold is hard-coded as `12.0` volts.
